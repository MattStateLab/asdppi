# ASD-PPI

This repository contains scripts and selected data files used to analyze the Psychiatric Cell Map Initiative (PCMI; https://pcmi.ucsf.edu/) ASD protein–protein interaction network generated by affinity purification–mass spectrometry (AP–MS) of 100 high-confidence ASD risk genes.

## Repository Layout

| Directory | Contents |
|---|---|
| `data/` | Shared ASD-PPI, hcASD102, and HEK293T proteome data used across analyses. |
| `asdppi_hek293T/` | HEK293T proteome expression analysis of bait, prey, and background proteins. |
| `risk_gene_enrichment/` | Fisher's exact test analyses for enrichment of ASD, DD, SCZ, and SFARI risk genes. |
| `ssc_enrichment/` | Satterstrom 2020 SSC de novo variant enrichment and downsampling analyses. |
| `bsRNAseq/` | BrainSpan developmental RNA-seq expression analyses. |
| `gtex/` | GTEx tissue expression analyses. |
| `organoid/` | FOXP1 organoid PIP-seq, CUT&Tag, and IHC analyses. |

Each analysis directory has its own `README.txt` with a file-level guide.

## Running Scripts

Run scripts from the repository root so relative paths resolve correctly. For example:

```r
source("data/asd102/download_satterstrom_hcASD102.R")
source("data/hek293tProteome/01_define_hek293tproteome.R")
source("bsRNAseq/scripts/02_asdPPI_bsRNAseq_prenatal_expression.R")
```

The numbered scripts generally follow the intended order within each analysis directory.
